Test case regeneration

Internship project (Linedata Tunisia)

Our solution consists in designing and developing two functionalities which will be implemented in the TAF360 tool, allowing essentially to manage error cases and fail cases. For fail cases, we will launch the script in replay mode. This feature will give the possibility to visualize the execution of a test case to help the user to better determine which part of it caused the execution to be interrupted the execution and send a more detailed report to the developers so that developers can fix the problem. For error cases, we will simply replace the current value with the reference value and force the execution to give a successful result
