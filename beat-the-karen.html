Beat The Karen

Internship project (Lanterns Studio)

Beat the Karen is a Beat them all game playable in Virtual Reality where you get to fight Karens without any consequences and using elements from the environment. The game will be set in multiple levels like clothing store,restaurant, parking lot...

Gitlab repository

Features that i worked on :

  • Grabbing objects
  • Procedural hand animation
  • Scoring system depending on the magnitude of the attack
  • Ragdoll effect
  • Attacking (Punching or with objects)
  • NPC IA
  • Implementing animations
