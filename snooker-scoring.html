Snooker Scoring

Extra-curricular project (Inceptum Junior Entreprise)

Snooker Scoring is a tool that manages the scoring system of the Snooker game and has been developed with Qt 5.9.9 and Qt Material Widgets in order to implement it in a RaspBerry card.

Github repository

Features that i worked on :

  • User interface
  • Frames management
  • Break management
  • Fault management
  • Scoring management
