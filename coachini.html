Coach'ini

Academic project

Elected at the project ball for the first year in the engineering cycle

Coach'ini is a multi-platform application that allows coaches to share their offers, experiences and skills with other users. On the other hand coach'ini allows clients to consult the offers of coaches in different categories and even to book coaching sessions. Our clients can also participate in events programmed by their coaches to make their sports adventure more enjoyable. Of course, our application includes a messaging system to facilitate communication between our clients and their coaches.

The desktop version of coach'ini was made with Java and Java FX, the web version is made with Symfony and the mobile with Java and Codename One, and the data in a SQL database.

Github repository

Features that i worked on :

  • Direct messaging (with Java Socket and Web Socket)
  • Friends management
  • Complaints management
  • Complaints type management
